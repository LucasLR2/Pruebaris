Pruebaris v1.0
Aplicación web educativa de preguntas y respuestas sobre programación, desarrollada completamente con IA.
Descripción
Pruebaris es un juego interactivo diseñado para aprender y practicar conceptos de programación a través de preguntas de opción múltiple. Incluye dos modos de juego y un sistema de logros para motivar el aprendizaje continuo.
Características

Sistema de perfiles: Creación de cuenta con avatar personalizable y guardado local
Modo Normal: 70 preguntas con 1 punto cada una, sin penalización por error
Modo Infinito: Preguntas aleatorias con puntuación variable (1-3 puntos) y penalización por respuestas incorrectas
Sistema de logros: 10 logros desbloqueables basados en progreso
Estadísticas detalladas: Seguimiento de partidas jugadas, puntos acumulados y precisión
Exportación de perfil: Guarda tu progreso en formato JSON

Contenido educativo
El banco de preguntas cubre los siguientes temas:

Programación Orientada a Objetos (POO)
Estructuras de datos (arrays, listas, pilas, colas)
Bases de datos y SQL
Redes y protocolos
Hardware y arquitectura de computadores

Instalación y uso

Clona o descarga el repositorio
Abre index.html en tu navegador web
Crea una cuenta nueva o carga un perfil existente
Selecciona un modo de juego y comienza a aprender

Tecnologías utilizadas

HTML5
CSS3 (con variables CSS y diseño responsive)
JavaScript vanilla
LocalStorage para persistencia de datos
API de Dicebear para avatares

Limitaciones conocidas

Los datos se almacenan localmente; limpiar el navegador eliminará el progreso
Se recomienda exportar el perfil regularmente para respaldo
No incluye funcionalidad multijugador ni tabla de clasificación global
